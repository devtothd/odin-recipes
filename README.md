# odin-recipes

This project was about making a recipes page with multiple recipes.

During this project I practiced:

- Linking pages to each other for better navigation between sites
- Adding boilerplates, images, alt attributes for accessibility reasons, lists, headers and paragraphs.
- Resizing images for better visual quality.
- While using git throughout the entire project, adding, committing, and pushing to main branch to practice workflow, while aiming for good and clear commit messages.
